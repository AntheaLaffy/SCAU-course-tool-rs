## 🛠 使用教程

### 使用前提

- 在教务系统的 **选课意向** 中已添加课程  
- 软件会自动抓包选课意向里的课程  
- 之后将进行 **全自动高速选课**

---

### 使用步骤

1. 在 **Release** 中下载压缩包，并 **完全解压**
2. 运行 `SCAU-auto-course.exe`
3. 回车，在弹出的浏览器窗口中登录账号和密码
4. 再次回车，输入持续循环提交的时间（单位：秒）
5. 程序开始执行脚本
6. 时间到达后程序自动退出

---

## ⚠️ 使用建议

- 建议在 **正式抢课前几分钟** 启动脚本
- 持续运行时间可以设置得稍长一些，以提高成功率
